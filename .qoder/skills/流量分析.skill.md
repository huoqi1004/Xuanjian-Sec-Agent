# 流量分析 (Traffic Analysis)

## 基本信息
- **ID**: traffic_analysis
- **类别**: 网络安全
- **图标**: 📊
- **版本**: 1.0.0
- **状态**: active

## 描述
分析网络流量，识别异常流量模式和潜在的安全威胁。支持实时流量分析和历史流量数据查询，可检测各种网络攻击和异常行为。

## 输入参数
| 参数名 | 类型 | 必填 | 默认值 | 描述 |
|--------|------|------|--------|------|
| `interface` | string | 是 | - | 网络接口名称，如`eth0`、`wlan0`等 |
| `filter` | string | 否 | "" | 流量过滤规则，使用BPF语法 |
| `duration` | int | 否 | 300 | 分析持续时间（秒），默认300秒 |
| `output_format` | string | 否 | "json" | 输出格式：`json`、`csv`、`report` |
| `save_results` | boolean | 否 | true | 是否保存分析结果到ELK |

## 输出结果
返回一个JSON对象，包含：
- `summary`: 流量摘要信息：
  - `total_packets`: 总数据包数
  - `total_bytes`: 总字节数
  - `duration`: 分析持续时间
  - `top_talkers`: 流量Top主机
- `anomalies`: 异常流量列表，每个异常包含：
  - `type`: 异常类型（如DDoS、端口扫描、异常连接等）
  - `severity`: 严重程度（high、medium、low）
  - `source_ip`: 源IP
  - `destination_ip`: 目标IP
  - `details`: 详细信息
  - `timestamp`: 时间戳
- `protocols`: 协议分布
- `top_ports`: 热门端口

## 调用方式
1. 使用Wireshark或Zeek捕获网络流量
2. 应用过滤规则（如果指定）
3. 分析流量模式和特征
4. 检测异常流量和潜在威胁
5. 生成分析报告
6. 可选：将结果保存到ELK Stack

## 依赖工具/API
- Wireshark（流量捕获和分析）
- Zeek（网络安全监控）
- ELK Stack（数据存储和可视化）
- 本地大模型（异常检测和分析）
